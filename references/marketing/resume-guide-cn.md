# 市场运营简历写作指南

> 来源：资深 Marketing / 运营 Hiring Manager 视角

## 核心原则

### 市场运营看简历的底层逻辑

当 Hiring Manager 筛选市场/运营简历时，内心判断标准是：

| 简历上的说法 | 面试官的实际质疑 |
|---|---|
| "负责品牌推广" | 品牌曝光量？认知度提升？多少预算？ |
| "运营微信公众号" | 粉丝量？打开率？转化率？涨粉来源？ |
| "做了增长" | 获客成本？留存率？LTV？增长了多少？ |
| "策划过活动" | 什么规模？ROI？拉新多少？收入多少？ |

**结论：** 不要写"负责运营""负责推广"这种职责描述。要写你**驱动了什么增长、花了多少预算、带来了多少转化、沉淀了什么方法论**。

### 万能公式

```
【业务指标】+【动作】+【量化结果】+【方法论沉淀】
```

## 各模块写法拆解

### 数字营销 / 增长

**差的写法：**
> 负责公司产品的新媒体运营，包括微信公众号、小红书、抖音等渠道的内容发布和粉丝增长。

**好的写法：**

```
增长营销 | 全年获客 12 万，CAC 从 ¥180 降至 ¥65，ROI 1:4.2
渠道矩阵：搭建"小红书种草 + 抖音短视频 + 信息流投放"链路，月均带来 3 万线索
内容增长：产出爆款笔记 23 篇（单篇最高曝光 580 万），自然流量占比从 28% 提升至 61%
投放优化：A/B 测试落地页 14 个版本，转化率从 3.2% 提升至 7.8%，月省投放预算 4.5 万
用户裂变：设计"邀请得会员"裂变活动，K 因子 1.32，30 天裂变新增 4.2 万用户
```

**着重写什么：**
- 结果前置：获客数、CAC、ROI、转化率、LTV
- 渠道能力：具体渠道名称 + 各渠道贡献 + 优化动作
- 增长方法论：裂变玩法、A/B 测试、归因模型

### 品牌营销

**差的写法：**
> 负责公司品牌宣传推广，制定品牌策略，维护品牌形象，提升品牌知名度。

**好的写法：**

```
品牌营销 | 年度品牌搜索量 +340%，品牌认知度调研得分 72→89
品牌战役：策划"XX 主题"年度Campaign，全渠道曝光 2.3 亿，微博热搜 3 次
KOL 矩阵：搭建 200+ KOL 资源库（头部/腰部/KOC 分层），单次合作 CPM 降低 40%
跨界合作：主导 X 品牌联名活动，双方新增粉丝合计 56 万，联名款售罄 12 分钟
媒体关系：维护 80+ 核心媒体关系，年度正面报道 120+，负面舆情 24h 响应率 100%
```

**着重写什么：**
- 品牌指标：搜索量、认知度、舆情正向率
- Campaign 规模：曝光量、互动量、热搜次数、媒体覆盖
- 资源沉淀：KOL 库、媒体关系、合作品牌

### 用户运营 / 社群运营

**差的写法：**
> 负责用户社群运营，维护用户关系，提升用户活跃度和留存率。

**好的写法：**

```
用户运营 | 管理 5 万+ 用户社群，30 日留存 41%，ARPU ¥128
分层运营：搭建 RFM 用户分层模型，高价值用户（Top 10%）贡献 62% 收入
社群 SOP：设计"新群7天"激活 SOP，新群首周活跃率从 23% 提升至 58%
用户活动：策划月度主题打卡活动，参与率 34%，UGC 内容 1200+ 条/月
流失预警：搭建流失预警模型（准确率 79%），挽回高价值用户 860 人，挽回收入 ¥32 万
```

**着重写什么：**
- 运营指标：留存率、ARPU、活跃度、流失率
- 分层方法论：RFM、用户画像、生命周期管理
- 工具/SOP：自动化触达、预警模型、SOP 流程

### 内容营销

**差的写法：**
> 负责公司产品的内容营销，包括文章撰写、视频制作、自媒体运营。

**好的写法：**

```
内容营销 | 全年产出 360 篇内容，全网曝光 8000 万，内容获客占比 35%
SEO 内容：产出 120 篇长尾词内容，自然搜索流量 +420%，月均有机流量 15 万
视频内容：抖音/视频号双平台运营，粉丝合计 28 万，单条视频最高播放 320 万
内容中台：搭建内容选题库（200+ 标签）+ 内容分发 SOP，人均产出效率 +180%
白皮书/报告：产出 3 份行业白皮书，下载量 4.2 万，MQL 转化 3800 条
```

**着重写什么：**
- 内容指标：曝光量、播放量、自然流量增长、内容获客占比
- 分发能力：各平台运营数据、粉丝增长、爆款率
- 体系化建设：内容中台、选题库、分发 SOP

### 产品营销 (Product Marketing)

**差的写法：**
> 负责产品上市推广，撰写产品卖点，配合销售团队进行产品培训。

**好的写法：**

```
产品营销 | 主导 4 个核心产品上市，首月 GMV 合计 ¥2800 万
Go-to-Market：制定 XX 产品 GTM 策略，上市首周注册用户 5 万，付费转化率 8.3%
销售赋能：制作产品 Battle Card + Demo Video，销售成单周期缩短 22%
竞品分析：输出季度竞品报告（覆盖 8 家竞品），定价策略调整带来毛利 +5.2%
客户案例：打造 12 个标杆客户案例，助力销售赢单率从 31% 提升至 44%
```

**着重写什么：**
- 上市成果：首月/首季收入、用户数、转化率
- 销售赋能：工具产出、成单周期、赢单率
- 市场洞察：竞品分析、定价策略、客户案例

## 三个不要写 vs 三个必须写

| 不要写（一眼识破） | 必须写（体现专业度） |
|---|---|
| "负责新媒体运营" | "全网粉丝 28 万，月均曝光 800 万，内容获客占比 35%" |
| "策划过线上活动" | "策划裂变活动，K 因子 1.32，30 天新增 4.2 万用户" |
| "提升品牌知名度" | "年度品牌搜索量 +340%，认知度调研得分 72→89" |

## 不同公司的侧重点

| 公司类型 | 看重什么 | 简历侧重点 |
|---|---|---|
| 互联网大厂 | 数据驱动、方法论沉淀、规模化 | 增长指标、数据工具、体系化建设 |
| 传统企业转型 | 从 0 到 1 能力、落地执行 | 渠道搭建、品牌起盘、资源开拓 |
| 跨境电商 | 海外渠道、投放能力、本地化 | Meta/Google Ads、本地化内容、ROI |
| SaaS / B2B | Lead 质量、销售协同、内容营销 | MQL/SQL 转化、白皮书、客户案例 |

## ATS 策略

简历可能经过机器筛选或 Hiring Manager 快速浏览：

1. 从对应模块的高频关键词中选 5-10 个与自己匹配的
2. 关键词要**自然融入**经历描述，不要堆砌在末尾
3. 文件名用 `姓名-职位.pdf`（不用 `.doc`，排版易错乱）
